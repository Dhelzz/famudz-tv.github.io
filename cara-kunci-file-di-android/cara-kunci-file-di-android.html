Cara Mengompres file ke zip dengan password di android

langkah-langkahnya

- Download ZArchiver Terlebih dahulu Di playstore,Kemudian.
- masuk ke ZArchiver kemudian2 pilih file atau folder.
- Untuk folder Tekan dan Tahan hingga muncul menu.
- Untuk file tinggal sekali pencet hingga muncul menu.
- Kemudian pilih kompres.
- Kemudian bisa ganti nama arsipnya.
- Kemudian bisa diganti Formatnya.
- Temudian masukan passwordnya misalnya "123".
- Terakhir pilih ok, dan file pun siap
Untuk file yang sudah di beri password tidak bisa di ekstrak dengan bawaan hp,.
Jadi hanya bisa di buka dengan ZAchiver atau pc. Dan itu juga membutuhkan Password.
